# earthquake-streamlit-dashboard
Global Seismic Trends 

## Overview
This project analyzes global earthquake data and presents key insights through
an interactive Streamlit dashboard. SQL queries are used to extract meaningful
patterns from structured data stored in a MySQL database.

---

## Problem Statement
Earthquake data is large and complex. This project aims to simplify analysis by
identifying trends, strong events, and risk-related patterns using data-driven
methods.

---

## Dataset
- Source: USGS Earthquake Dataset
- Includes magnitude, depth, location, time, and tsunami indicators

---

## Tools & Technologies
- Python
- MySQL
- Streamlit
- Pandas
  

---

## Key Features
- Top 10 strongest earthquakes
- Year-wise earthquake trends
- Depth vs magnitude analysis
- Tsunami vs non-tsunami comparison
- SQL-based data analysis
